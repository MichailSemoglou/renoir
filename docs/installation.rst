Installation
============

Basic Installation
------------------

.. code-block:: bash

   pip install renoir-wikiart

With Visualization Support (Recommended)
-----------------------------------------

.. code-block:: bash

   pip install 'renoir-wikiart[visualization]'

From Source
-----------

.. code-block:: bash

   git clone https://github.com/MichailSemoglou/renoir.git
   cd renoir
   pip install -e '.[visualization]'

Requirements
------------

- Python 3.9+
- numpy, Pillow, scikit-learn, pandas, scipy, datasets
- matplotlib, seaborn (optional, for visualizations)
